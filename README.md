This project aims to track Natural Language Processing (NLP) resources for the Arabic language and gives an overview
of the state-of-the-art (SOTA) across the most common NLP tasks and their corresponding datasets.

The idea is to track papers, datasets, pre-trained models and SOTA of each NLP task. It's built with **business applications** in mind, the objective being to have a starting point for NLP based solutions, rather than fully focused on research.

The main objective is to provide the reader with a quick overview of benchmark datasets and the state-of-the-art for their
task of interest, which helps building a quick POC for their target solution.
